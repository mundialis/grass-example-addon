DESCRIPTION

v.example is an example for a GRASS GIS addon

EXAMPLE

Example 1

v.example output=tiles box=1000,1000

Create grid for current region with grid box size of 1000 x 1000 and output basename "tiles" and extract tiles only for a given polygon "poly_aoi".

v.example output=tiles box=1000,1000 polygon_aoi=poly_aoi
